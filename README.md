# Making SuperMe Better and the Network Stronger

A one-page proof-of-work artifact for SuperMe.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Deploy on Vercel

Deploy this `superme-30-day-plan/` directory as the project root in Vercel.

The page is built with Next.js, React, and Tailwind CSS. It has no backend and no environment variables.
